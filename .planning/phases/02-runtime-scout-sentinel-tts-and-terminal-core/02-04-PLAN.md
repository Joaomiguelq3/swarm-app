  <action>
    Create `scripts/smoke-pty.js` using `node-pty`. Spawn PowerShell on Windows with a short command that prints a known marker such as `SWARM_PTY_OK` and exits.     Do not implement renderer terminal panes or xterm.js in this plan.
  </action>
  <verify>
    Run `node --check scripts/smoke-pty.js`.
    Run `npm.cmd run smoke:pty`.
  </verify>
  <acceptance_criteria>
    - PTY smoke script receives streamed output.
    - Script exits cleanly after process completion.
  </acceptance_criteria>
</task>
